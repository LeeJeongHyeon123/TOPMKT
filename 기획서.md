## 🧑‍🤝‍🧑 1. 회원 시스템

### 회원가입 / 로그인
- 휴대폰 번호 인증 기반 회원가입 및 로그인
- 인증된 전화번호 = 로그인 ID (비밀번호 없음)
- 로그인 시 DB에 로그인 이력 저장 (ID, 시간, IP)
- 로그인 후 장기 세션 유지 (세부 정책은 기본정책.md 참조)
- 회원가입 시 `국가` 지정
- `소개글`은 회원가입 후 입력 받자. (선택 사항)(회원 가입 후 입력 및 수정 가능)(회원가입 후와 메인 페이지 진입시 프로필 정보를 입력하지 않은 것이 있으면 입력하러 갈건지 유도하는 팝업 출력)

### 사용자 프로필
- 항목: 프로필 사진, 닉네임(중복 불가), 소속 회사, 포지션(리더/판매원), 이메일 주소
- 휴대폰 번호와 닉네임만 필수 입력
- 휴대폰 번호는 수정 불가, 그 외 정보는 수정 가능
- 국가 변경 가능
- 프로필 미작성 상태로 메인 페이지에 접속할 때도 주기적으로 프로필 작성 유도

### 프로필 이미지 및 회사/자기소개 관리 [구현 완료]
- 프로필 이미지는 Firebase Storage에 저장하고 URL만 DB에 저장
- 이미지 최대 용량: 5MB, 지원 포맷: JPEG, PNG, GIF, WEBP
- 회사명: 최대 100자까지 입력 가능 (선택사항)
- 자기소개: 최대 5000자까지 입력 가능 (선택사항)
- 전용 프로필 설정 페이지 구현 (`/profile-settings.php`)
- 관련 API 엔드포인트:
  - 프로필 이미지 업로드: `/api/user/upload-profile-image.php`
  - 프로필 정보 업데이트: `/api/user/update-profile.php`
  - 내 프로필 정보 조회: `/api/user/profile.php`
  - 다른 회원 프로필 조회: `/api/user/get-user-profile.php`

## 🏢 2. 회사/비전 소개 게시판

- 본인이 속한 회사의 비전 소개 등록 가능 (예: 애터미, 뉴스킨 등)
- 첨부: 이미지, 동영상, 유튜브 링크, 텍스트
- 이미지/영상은 AI로 판단하여 적절하게 압축
- 좋아요, 댓글 기능
- 공유하기 (URL 복사 기능)
- 조회수, 좋아요 수 표시
- 태그 기능 + 태그 기반 검색 가능

## 🏆 15. 추천 리더 홍보 공간

- 메인 페이지 섹션에 랭킹 기반 추천 리더 노출
- 점수 산정 방식:
  - 게시글 작성: 10점
  - 댓글 작성: 1점
  - 좋아요 받기: 1점
- 월요일~일요일 기준으로 점수 집계 → 금주 추천 리더 선정
- 카드 UI: 프로필 + 점수 + "채팅하기" 버튼 포함
- 관리자 고정 상단 설정 가능
- 1위~100위까지 자동 롤링으로 노출
